## 商城系统管理后台

### 登录角色分为两种

- 超级管理员
- 普通商家

#### 超级管理员

##### 拥有的权限

- 查看整个系统的客流量，访客数量，男女分布，地区分布，下单情况，以及浏览最多的商品类型
- 查看所有店铺的流量，分析那个店铺访问量大（埋点）
- 查看所有用户的信息，不包括密码，但是提供用户修改密码功能
- 首页店铺推荐功能
- 首页店铺上轮播图功能
- 查看店铺售卖情况，以及店铺内售卖的商品
- 封号，封店铺（软删除）
- 首页模板的选择
- 想到了再加

#### 普通商家

- 修改新增商品
- 美化自己商品详情页面
- 查看商品收藏，点赞，评论数量
- 对付款了的商品进行发货
- 查看物流
- 查看订单号
- 查看营业额并且进行提现操作
- 店铺人员访问量查看
- 商品加入购物车情况展示
- 增加推荐的商品
- 更改商品展示顺序
